1. OpenCV(Open Source Computer Vision Library)
- 컴퓨터 비전과 이미지 처리를 위한 오픈소스 라이브러리
컴퓨터 비전: 인공지능의 한 분야. 컴퓨터와 시스템을 통해 디지털 이미지, 비디오 및 기타 시각적 입력에서 의미 있는 정보를 추출한 다음 이러한 정보를 바탕으로 작업을 실행
- 1999년 Intel에서 영상처리 기술을 개발하기 위한 목적
- 2000년 BSD 라이센스로 배포
- 2011년 이후 OpenCV2로 개발 시작


2. OpenCV 설치
(주피터나 코랩): pip install opencv-python
(파이참): 메뉴(menu) > 설정(setting) > 프로젝트(project) 이름 > python interpreter > '+' 클릭 후 opencv-python 검색 > install package 클릭

3. 컴퓨터 비전(Computer Vision)
- 영상 처리, 동영상 처리, 객체 탐지, 패턴 인식, 광학 문자 인식 등에서 활용

- 영상(Image)
픽셀(pixel): 이미지를 구성하는 가장 작은 단위
영상: 픽셀이 바둑판 모양의 격자에 나열되어 있는 형태. 2차원 행렬
그레이스케일 영상
    - 흑백 사진처럼 색상 정보가 없는 영상
    - 밝기 정보만으로 구성된 영상
    - 밝기 정보는 256단계로 표현(0 ~ 255)
    - numpy.uint8 = 8bit = 1byte
    - 가로크기 * 세로크기 = 28 * 28 = 784 bytes
트루컬러 영상
    - 컬러 사진처럼 색상 정보를 가지고 있기 때문에 다양한 색상을 표현할 수 있는 영상
    - red, green, blue 색 성분을 사용하고 각 256단계로 표현
    - 픽셀의 표현 -> 튜플 형태로 표현 -> (255, 255, 255)
    - numpy.ndarray = 3byte
    - 가로크기 x 세로크기 x 3 = 28 x 28 x 3 = 2352 byte
영상 파일 형식
    - bmp
        - 픽셀 데이터를 압축하지 않고 그대로 저장
        - 용량이 매우 큼(픽셀의 일반적인 크기)
        - 파일 구조가 단순해서 별도의 라이브러리 없이 프로그래밍이 가능
    - jpg, jpeg
        - 압축률이 좋아서 파일 용량이 크게 감소
        - 사진과 같은 컬러영상을 저장
        - 손실 압축
    - gif
        - 움직이는 영상 지원
        - 256색 이하의 영상을 저장
        - 무손실 압축
    - png
        - 웹 이미지용으로 권장
        - 무손실, 손실 압축
        - 알파 채널(투명도)을 지원, (255, 255, 255, 1)

4. 영상의 출력
영상 읽기: cv2.imread(영상, 방법)
      예) cv2.imread('./dog.bmp', cv2.IMREAD_GRAYSCALE)
      IMREAD_GRAYSCALE: 그레이스케일로 읽기
      IMREAD_COLOR: 트루컬러로 읽기(기본값, 불러온 영상의 색상 정보는 BGR 순서(RGB가 아님))
BGR을 RGB로 변환하는 방법(채널 순서 변환): cv2.cvtColor(영상, 변환방법)
    cv2.COLOR_BGR2RGB: BGR을 RGB로 변환
    cv2.COLOR_RGB2BGR: RGB를 BGR로 변환

5. 차원
변수명 = cv.imread(영상, 방법)
type(변수명): <class 'numpy.ndarray'>
변수명.shape: (h ,w) -> (높이, 너비) 또는 (h, w, 3) -> (높이 너비, 색상 채널수)
변수명.dtype: uint8

6. 도형 그리기
직선: cv2.line(영상, 직선의 시작점과 끝점 튜플, 선색상, 선두께, 선타입)
사각형: cv2.rectangle(영상, 사각형 꼭지점 좌표 튜플, 선색상, 선두께)
    ✔ 선두께를 -1로 설정하면 사각형 내부를 색상으로 채움
원: cv2.circle(영상, 원의 중앙좌표, 반지름 ,선색상, 선두께)
    ✔ 선두께를 -1로 설정하면 원 내부를 색상으로 채움
문자열: cv2.putText(영상, 문자열, 영상에서 문자열을 출력할 위치 좌표 튜플, 글꼴, 폰트크기, 색상)
